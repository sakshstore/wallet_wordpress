Saksh Wallet is a multi currency wallet you can use to store customer balance in any currency and it send /receive payment.

With details api it is easy to integrate this api from any plugin etc.

== Description ==

Wallet allows customers to store their money in a digital wallet. The admin can process refund to customer wallet. It is easy to send/receive/store payment and user balance.



== Installation ==

1. Download and extract   to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. "Dashboard"->"Account"
4. There are some examples on the settings page,
